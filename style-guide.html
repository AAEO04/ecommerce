‚ú® Design Principles

‚úÖ Do

  • Use white as dominant background (80%+)
  • Apply green to "Add to Cart" CTAs
  • Use purple for all links
  • Reserve red for urgency and sales
  • Maintain generous whitespace
  • Ensure color contrast meets WCAG AA

‚ùå Don't

  • Mix all brand colors on one component
  • Use colored backgrounds for large sections
  • Apply red to non-urgent actions
  • Reduce whitespace to fit more content
  • Use color as the only state indicator
  • Forget hover and focus states
